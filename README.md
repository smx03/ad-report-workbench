# 豆包联盟日报工作台

纯前端的广告日报与时报生成工具，可通过 GitHub Pages 访问。

## 日报流程

1. 上传日报日、前一日、七日前三份全部账户 Excel。
2. 上传本次使用的最新账户匹配表。
3. 浏览器本地解析、校验、计算并生成拉新/卸载两张 PNG。

四份 Excel 不会发送给 GitHub 或任何后端。原始文件仅存在于当前页面内存，刷新或关闭页面后释放。

## 时报流程

上传一份包含今天、昨天、前天分时数据的 Excel，选择发送时刻后生成两张时报图片。该流程同样只在浏览器本地运行。

## 本地预览

```bash
pnpm preview
```

打开 `http://127.0.0.1:4319/`。开发时使用 HTTP 预览，不要直接双击 `index.html`。

## 发布

`.github/workflows/pages.yml` 会在 `main` 分支更新后将 `public/` 发布到 GitHub Pages。

## 安全边界

- 仓库不包含账户匹配表、原始数据、SQLite、历史日报或上传文件。
- 页面只托管 HTML、CSS、JavaScript 和浏览器版 ExcelJS。
- 纯前端规则会下载到访问者浏览器，因此计算参数不属于服务器密钥。
